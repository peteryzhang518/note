# vim terminal
进入terminal
```
:terminal
```
退出终端
在终端窗口中输入 exit 或按 Ctrl-D 退出当前的终端会话
切换终端
按 Ctrl-W N（在终端模式下）可以切换回正常的 Vim 模式（并不关闭终端窗口），可以继续在 Vim 中编辑文件。
切换回终端
切换回终端模式：如果你在 Vim 编辑模式下，想回到终端模式，可以按 Ctrl-W W 来切换焦点回到终端窗口。
